# Forever E-Commerce Frontend

Forever E-Commerce is a modern and feature-rich online shopping platform frontend, designed to provide a seamless and responsive shopping experience for users. Built with **React** and **Vite**, this project showcases clean design, interactivity, and essential e-commerce features.

## Features

- **Product Listing**: Browse products with detailed views.
- **Category Filtering**: View products based on selected categories.
- **Responsive Design**: Optimized for mobile, tablet, and desktop devices.

## Tech Stack
- **Frontend Framework**: React (with Vite for faster development).
- **UI Components**: React-tailwind for consistent and clean design.
- **State Management**: React hooks for managing application state.
- **Styling**: CSS for custom font for typography.
